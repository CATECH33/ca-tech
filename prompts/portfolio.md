# Prompt — Page Portfolio / Réalisations CA-TECH

## Objectif
Prouver l'expertise de CA-TECH par des réalisations concrètes et inspirer confiance.

## Structure de la page

### Header
- Titre : "Nos Réalisations"
- Sous-titre : "Des projets conçus pour performer"

### Filtres
- Tous
- Sites vitrines
- E-commerce
- Applications
- Identités visuelles
- IA & Automatisation

### Grille de projets
Chaque carte contient :
- Capture / visuel du projet
- Nom du projet
- Type (badge)
- Description courte (1-2 lignes)
- Technologies utilisées
- Lien "Voir le projet" (si disponible)

### CTA final
- "Votre projet pourrait être ici."
- Bouton : "Démarrer mon projet"

## Ton
Sobre, professionnel. Laisser les réalisations parler.

## Règles
- Grille masonry ou 3 colonnes
- Hover : overlay avec infos du projet
- Transitions douces
- Mobile First : 1 colonne sur mobile
