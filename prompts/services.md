# Prompt — Page Services CA-TECH

## Objectif
Présenter chaque service clairement et inciter le visiteur à demander un devis.

## Structure de la page

### Header
- Titre : "Nos Services"
- Sous-titre : "Des solutions tech complètes pour votre entreprise"

### Grille de services

#### Support Informatique
- Description : Assistance à distance et sur site pour vos équipements et logiciels
- Cible : PME, artisans, cabinets
- CTA : "Demander une assistance"

#### Maintenance
- Description : Maintenance préventive et corrective pour assurer la continuité de votre activité
- Cible : Toute structure avec parc informatique
- CTA : "Souscrire un contrat"

#### Réseaux
- Description : Installation, administration et sécurisation de vos réseaux et systèmes
- Cible : PME, associations, cabinets
- CTA : "Audit réseau gratuit"

#### Développement Web
- Description : Sites vitrines, e-commerce et applications web sur mesure
- Cible : Commerçants, artisans, PME
- CTA : "Voir nos réalisations"

#### Intelligence Artificielle
- Description : Agents IA et automatisation pour optimiser vos processus métier
- Cible : PME, toute structure souhaitant gagner du temps
- CTA : "Découvrir nos solutions IA"

#### Formation
- Description : Formations Web et IA adaptées à votre niveau et vos besoins
- Cible : Équipes, indépendants, dirigeants
- CTA : "Voir les formations"

### CTA global
- "Vous ne savez pas par où commencer ? Contactez-nous."
- Bouton : "Demander un devis gratuit"

## Ton
Clair, rassurant, expert. Une service = un bénéfice concret.

## Règles
- Cartes avec glassmorphism
- Icône par service
- Mobile First
