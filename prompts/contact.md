# Prompt — Page Contact CA-TECH

## Objectif
Maximiser les demandes de devis et faciliter la prise de contact.

## Structure de la page

### Header
- Titre : "Contactez-nous"
- Sous-titre : "Un projet ? Une question ? On vous répond rapidement."

### Formulaire de contact
Champs :
- Nom complet (requis)
- Email (requis)
- Téléphone (optionnel)
- Entreprise (optionnel)
- Type de service souhaité (liste déroulante : Support, Maintenance, Réseaux, Web, IA, Formation, Autre)
- Message / Description du projet (requis)
- Bouton : "Envoyer ma demande"

### Informations de contact
- Email : contact@ca-tech.fr (à adapter)
- Réponse garantie sous 24h ouvrées

### Réassurance
- "Devis gratuit et sans engagement"
- "Réponse sous 24h"
- "Accompagnement personnalisé"

## Ton
Accessible, rassurant. Lever les freins à la prise de contact.

## Règles
- Formulaire épuré, sans friction
- Validation en temps réel des champs
- Message de confirmation après envoi
- Mobile First
