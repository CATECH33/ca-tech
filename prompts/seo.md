# Prompt — SEO CA-TECH

## Objectif
Optimiser chaque page pour le référencement naturel et générer du trafic qualifié.

## Mots-clés principaux
- agence web
- création site internet
- création site e-commerce
- intelligence artificielle entreprise
- automatisation entreprise
- création logo
- identité visuelle
- support informatique PME

## Mots-clés secondaires / longue traîne
- agence web pour PME
- créer un site vitrine professionnel
- développement web sur mesure
- agent IA pour entreprise
- automatisation processus métier
- formation intelligence artificielle
- maintenance informatique entreprise
- identité visuelle pour artisan

## Règles SEO par page

### Balises meta
- Title : 50-60 caractères, mot-clé principal en premier
- Description : 150-160 caractères, incitative, avec CTA

### Contenu
- 1 seul H1 par page
- H2 et H3 structurés avec mots-clés secondaires
- Minimum 300 mots par page
- Contenu original, sans duplication

### Images
- Alt text descriptif sur chaque image
- Noms de fichiers explicites (ex : creation-site-internet-pme.jpg)
- Format WebP, poids < 100ko

### Performance
- Score Lighthouse > 90
- LCP < 2.5s
- CLS < 0.1

### Technique
- Sitemap XML généré et soumis à Google
- Robots.txt configuré
- Balises Open Graph pour le partage social
- Schema.org : LocalBusiness, Service, WebSite

## Suivi
- Google Search Console connecté
- Google Analytics 4 configuré
- Rapport mensuel des positions
