# Prompt — Page Applications CA-TECH

## Objectif
Présenter les applications développées par CA-TECH et générer de l'intérêt / des inscriptions.

## Structure de la page

### Header
- Titre : "Nos Applications"
- Sous-titre : "Des outils pensés pour simplifier votre quotidien"

### Cartes applications

#### PAS MAL
- Type : Marketplace de services
- Description : Trouvez et réservez des prestataires qualifiés en quelques clics
- Statut : (En développement / Disponible)
- CTA : "En savoir plus" ou "Rejoindre la bêta"

#### PEMOUS MONEY
- Type : Gestion financière
- Description : Suivez vos dépenses, gérez votre budget et pilotez vos finances simplement
- Statut : (En développement / Disponible)
- CTA : "En savoir plus" ou "Essayer gratuitement"

#### CV MAGIC
- Type : Création de CV assistée par IA
- Description : Créez un CV professionnel et percutant en moins de 5 minutes grâce à l'IA
- Statut : (En développement / Disponible)
- CTA : "Créer mon CV"

#### CA-TECH Manager
- Type : Gestion d'entreprise
- Description : Pilotez votre activité, vos clients et vos projets depuis une seule plateforme
- Statut : (En développement / Disponible)
- CTA : "Demander un accès"

### CTA final
- "Une idée d'application ? On peut la construire ensemble."
- Bouton : "Nous contacter"

## Ton
Innovant, enthousiaste. Mettre en avant la valeur de chaque outil.

## Règles
- Cartes avec glassmorphism
- Badge de statut (En développement / Disponible / Bêta)
- Mobile First
