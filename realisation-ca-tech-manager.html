CA-TECH Manager
ERP & gestion interne

CA-TECH Manager est l'outil de gestion interne que nous avons développé pour piloter notre propre activité d'agence. CRM, projets, devis, factures, agenda, analytics — tout en un seul endroit, avec des données en temps réel. Développé avec React, TypeScript et Supabase.

Le projet

Développer l'outil que
les SaaS existants ne donnaient pas

Gérer une agence web avec un assemblage de Notion, Google Sheets, une boîte mail et un outil de facturation séparé n'est pas tenable à scale. Nous avons décidé de développer notre propre outil de gestion — adapté exactement à notre façon de travailler, pas à un workflow générique imaginé par un éditeur de logiciels.

Le CA-TECH Manager centralise l'ensemble du pilotage de l'agence : suivi des clients, gestion des projets par étapes, création et envoi de devis et factures, agenda partagé, suivi du temps, et tableaux de bord analytiques en temps réel. Tout ce dont une agence web a besoin, sans la complexité inutile des outils tout-en-un du marché.

Stack technique

L'application a été développée avec un stack moderne, performant et maintenable :

  • Frontend : React 19 + TypeScript pour une UI typée et robuste
  • Styling : TailwindCSS pour un développement UI rapide et cohérent
  • State management : TanStack Query v5 pour la gestion des données asynchrones
  • Backend : Supabase (PostgreSQL) pour la base de données, l'authentification et les fonctions Edge
  • Temps réel : Supabase Realtime pour les notifications et mises à jour live
  • Notifications : Email (Resend), Telegram — depuis les Edge Functions, zéro clé API côté client

Les modules développés

L'application comprend 8 modules principaux, chacun correspondant à un besoin opérationnel précis de l'agence. Chaque module a été conçu pour être simple à utiliser et rapide à accéder — l'efficacité opérationnelle était la priorité, pas l'exhaustivité des fonctionnalités.

Stack & technologies

  • Frontend : React 19, TypeScript
  • CSS : TailwindCSS
  • Requêtes : TanStack Query v5
  • Routing : React Router v7
  • Backend : Supabase (PostgreSQL)
  • Auth : Supabase Auth
  • Fonctions : Supabase Edge Functions (Deno)
  • Notifications : Resend (email) + Telegram Bot API
  • Build : Vite
  • Déploiement : Vercel

Les modules

8 modules pour piloter
l'agence de A à Z

CRM Clients

Fiche client complète : coordonnées, historique des projets, devis et factures associés, notes internes. Recherche et filtres avancés. 127 clients actifs gérés.

Gestion Projets

Suivi des projets par étapes personnalisables, avancement en pourcentage, dates d'échéance, assignation à un client. 42 projets actifs en temps réel.

Devis & Factures

Création de devis et factures avec calcul automatique des totaux HT/TVA/TTC, gestion des statuts (brouillon, envoyé, accepté, payé), export PDF.

Agenda partagé

Calendrier des rendez-vous clients, jalons de projets et événements internes. Vue semaine/mois avec code couleur par type d'événement.

Analytics & KPIs

Tableau de bord avec CA mensuel, nombre de clients actifs, projets en cours, factures en attente. Graphiques en temps réel mis à jour à chaque nouvelle facture.

Notifications Temps Réel

Système de notifications multi-canaux : in-app (Supabase Realtime), email (Resend) et Telegram pour les événements critiques. Tout déclenché depuis les Edge Functions — zéro clé API côté client.
