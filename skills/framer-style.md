# Framer Style Skill

CA-TECH s'inspire de l'esthétique Framer — des interfaces qui semblent vivantes et interactives.

## Esthétique Framer

- Dark mode par défaut avec des accents lumineux
- Surfaces vitrées (glassmorphism subtil, pas excessif)
- Micro-animations omniprésentes mais discrètes
- Composants qui réagissent au hover et au scroll
- Feeling "app" plutôt que "site web"

## Éléments visuels

- Cartes avec backdrop-filter blur et bordures semi-transparentes
- Badges et pills pour catégoriser (petits, colorés, arrondis)
- Boutons avec inner glow et transition de couleur
- Icônes outlined, fines, monochromes
- Dividers subtils — lignes gradient qui s'estompent aux bords

## Motion design

- Spring animations (cubic-bezier rapide, léger overshoot)
- Stagger sur les listes — chaque élément apparaît avec un délai
- Scale + opacity sur les hover cards
- Smooth scroll entre sections
- Cursor-follow effects si pertinent (sobre)

## Layout

- Grilles modulaires avec gaps généreux
- Sections alternées (fond légèrement différent)
- Conteneurs étroits pour le texte (max 600px)
- Conteneurs larges pour les grilles (max 1200px)
- Padding vertical conséquent entre sections (100px+)

## Palette type

- Background : #050d1a à #0f1d32
- Surface : rgba(255,255,255, 0.03–0.06)
- Bordures : rgba(255,255,255, 0.06–0.12)
- Texte principal : #ffffff
- Texte secondaire : rgba(255,255,255, 0.6)
- Texte tertiaire : rgba(255,255,255, 0.4)
- Accent : #0066FF → #00D4FF (gradient)

## Anti-patterns

- Pas de néon agressif
- Pas de gradients arc-en-ciel
- Pas d'ombres portées lourdes
- Pas de textures ou patterns de fond visibles
- Pas de parallax excessif
