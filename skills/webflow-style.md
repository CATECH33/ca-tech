# Webflow Style Skill

CA-TECH s'inspire du niveau de qualité des sites construits sur Webflow par les meilleures agences.

## Caractéristiques Webflow

- Layouts asymétriques maîtrisés — pas de grilles rigides 12 colonnes
- Grandes typographies expressives (clamp responsive)
- Sections pleine largeur avec conteneurs internes
- Scroll animations natives — parallax subtil, fade-in, slide-up
- Sticky sections et horizontal scroll quand pertinent

## Patterns de design

- Hero split : texte à gauche, visuel à droite (ou inversé)
- Cards avec hover states élaborés (scale, shadow, border glow)
- Marquee/ticker pour les logos clients ou technologies
- Grilles Bento pour présenter des features
- Mega-footers avec navigation complète

## Typographie

- Titres : taille généreuse (clamp entre 2rem et 5rem)
- Letterspacing négatif sur les gros titres
- Line-height serré sur les headlines (1.05–1.15)
- Contraste de poids : 900 pour les titres, 400 pour le corps
- Utilisation de gradient text pour les mots-clés

## Animations

- Scroll-triggered reveals (IntersectionObserver)
- Transitions CSS fluides (300–500ms, ease-out)
- Hover states réactifs mais pas envahissants
- Pas de bounce, pas de flash — tout est smooth
- Loading states élégants (skeleton, blur-up)

## Responsive

- Mobile-first obligatoire
- Les layouts se simplifient, pas se compriment
- Les images s'adaptent, les textes restent lisibles
- Navigation mobile : drawer ou fullscreen overlay
