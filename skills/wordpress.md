# WordPress Skill — CA-TECH

## Objectif
Développer des sites WordPress performants, sécurisés et maintenables.

## Types de projets

- Sites vitrines professionnels
- Blogs et magazines en ligne
- Sites e-commerce (WooCommerce)
- Sites institutionnels multi-pages
- Landing pages

## Stack recommandé

- **Thème** : thème custom ou starter theme (Underscores, Sage)
- **Page builder** : Elementor Pro ou Gutenberg natif (FSE)
- **E-commerce** : WooCommerce
- **Forms** : Gravity Forms ou WPForms
- **SEO** : Yoast SEO ou Rank Math
- **Cache** : WP Rocket ou LiteSpeed Cache
- **Sécurité** : Wordfence ou Sucuri

## Bonnes pratiques

### Performance
- Hébergement optimisé WordPress (Kinsta, WP Engine, o2switch)
- Images WebP avec plugin (ShortPixel, Imagify)
- Lazy loading natif
- Minification CSS/JS
- CDN (Cloudflare)
- Base de données optimisée régulièrement

### Sécurité
- Mises à jour WordPress, thème et plugins régulières
- Authentification 2FA pour admin
- Limiter les tentatives de connexion
- Sauvegardes automatiques (UpdraftPlus)
- Certificat SSL obligatoire
- Masquer la version WordPress
- Préfixe de table personnalisé

### SEO
- Permaliens personnalisés (/%postname%/)
- Sitemap XML automatique
- Breadcrumbs
- Schema.org via plugin ou code
- Temps de chargement < 3s

## Développement custom

- Child theme pour personnalisations
- Custom Post Types pour contenus spécifiques
- ACF (Advanced Custom Fields) pour champs personnalisés
- Hooks (actions + filters) plutôt que modifier le core
- REST API pour intégrations headless

## Livrables

- Site WordPress installé et configuré
- Thème personnalisé aux couleurs de la marque
- Formation client (back-office)
- Documentation de maintenance
- Sauvegardes configurées
