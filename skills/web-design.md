# Web Design Skill — CA-TECH

## Objectif
Créer des sites web visuellement impactants, performants et accessibles.

## Principes

- Mobile First : conception prioritaire pour écrans mobiles
- Performance First : score Lighthouse > 90
- Design Premium : inspiré Apple, Stripe, Vercel, Linear
- Accessibilité WCAG AA minimum

## Stack technique

- HTML5 sémantique
- CSS3 (Custom Properties, Grid, Flexbox, animations)
- JavaScript vanilla ou frameworks modernes
- Responsive design avec breakpoints : 480px, 768px, 1024px, 1440px

## Livrables

- Maquettes desktop + mobile
- Code source optimisé
- Assets exportés (images WebP/AVIF, icônes SVG)
- Documentation technique

## Checklist qualité

- Validation W3C HTML/CSS
- Tests cross-browser (Chrome, Firefox, Safari, Edge)
- Tests responsive sur vrais appareils
- Temps de chargement < 3s
- Pas de CLS (Cumulative Layout Shift)
- Images lazy-loaded
- Fonts optimisées (preload, font-display: swap)

## Animations

- Transitions CSS légères (300ms max)
- IntersectionObserver pour scroll reveal
- Pas d'animation qui bloque le thread principal
- Respecter prefers-reduced-motion
