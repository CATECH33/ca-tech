# UI/UX Design Skill — CA-TECH

## Objectif
Concevoir des interfaces utilisateur intuitives et des expériences mémorables.

## Principes UX

- Hiérarchie visuelle claire
- Maximum 3 clics pour atteindre n'importe quelle information
- Feedback immédiat sur chaque interaction
- Cohérence visuelle sur toutes les pages
- Formulaires courts et progressifs (multi-step si nécessaire)

## Principes UI

- Espacement généreux (8px grid system)
- Typographie lisible (taille min 16px sur mobile)
- Contraste suffisant (ratio 4.5:1 minimum)
- Composants réutilisables et cohérents
- Dark mode par défaut pour CA-TECH

## Palette d'interactions

- Hover : transition douce (200-300ms)
- Focus : outline visible pour accessibilité
- Active : feedback visuel immédiat
- Loading : skeleton screens ou spinners subtils
- Erreur : messages clairs, couleur rouge doux (#ef4444)
- Succès : couleur verte (#22c55e)

## Outils

- Figma pour maquettes et prototypage
- Design tokens pour cohérence cross-platform

## Livrables

- Wireframes basse fidélité
- Maquettes haute fidélité (desktop + mobile)
- Prototype interactif
- Design system / composants documentés
- Spécifications pour développeurs
