# SEO Skill — CA-TECH

## Objectif
Optimiser le référencement naturel pour maximiser la visibilité sur les moteurs de recherche.

## SEO Technique

- HTML sémantique (h1 unique, h2-h6 hiérarchiques)
- Balises meta (title < 60 chars, description < 160 chars)
- URLs propres et descriptives
- Sitemap XML + robots.txt
- Schema.org / JSON-LD (LocalBusiness, Organization, FAQ)
- Canonical URLs
- Hreflang pour sites multilingues

## Performance

- Core Web Vitals : LCP < 2.5s, FID < 100ms, CLS < 0.1
- Images optimisées (WebP/AVIF, lazy loading, srcset)
- CSS/JS minifiés et compressés (gzip/brotli)
- Cache navigateur configuré
- CDN recommandé

## Contenu

- Mots-clés recherchés et positionnés naturellement
- Balises alt sur toutes les images
- Liens internes stratégiques
- Contenu structuré avec titres clairs
- FAQ riche pour featured snippets

## SEO Local (CA-TECH)

- Google Business Profile optimisé pour chaque ville (Paris, Lyon, Dijon, Troyes)
- NAP cohérent (Nom, Adresse, Téléphone)
- Avis clients encouragés
- Pages locales dédiées

## Outils de suivi

- Google Search Console
- Google Analytics 4
- Lighthouse audits réguliers

## Checklist avant mise en ligne

- Toutes les pages ont un title et meta description uniques
- Aucune page orpheline
- Aucun lien cassé (404)
- Redirections 301 en place si migration
- Open Graph et Twitter Cards configurés
