# Workflow Design — CA-TECH

## Vue d'ensemble
Processus de création graphique pour logos, identités visuelles, flyers et supports de communication.

## Phases

### Phase 1 — Brief créatif (Jour 1-2)
- Analyse du brief client (voir process-client.md)
- Recherche sectorielle et concurrentielle
- Création d'un moodboard (couleurs, styles, inspirations)
- Validation du moodboard avec le client

### Phase 2 — Exploration (Jour 3-5)
- Esquisses rapides (croquis, concepts)
- Minimum 3 directions créatives distinctes
- Chaque direction accompagnée d'une justification
- Présentation au client avec contexte d'utilisation

### Phase 3 — Développement (Jour 6-8)
- Affinage de la direction validée
- Création des déclinaisons :
  - Version principale
  - Version monochrome
  - Version négative (fond sombre)
  - Icône / favicon
- Tests de lisibilité à différentes tailles
- Tests sur différents fonds

### Phase 4 — Révisions (Jour 9-10)
- Intégration des retours client (tour 1)
- Ajustements fins (couleurs, proportions, typographie)
- Intégration des retours client (tour 2 si nécessaire)
- Validation finale écrite

### Phase 5 — Finalisation & Export (Jour 11-12)
- Export dans tous les formats requis :
  - SVG (vectoriel optimisé)
  - PNG (transparent, 1x / 2x / 3x)
  - PDF (CMYK, print-ready)
  - ICO (favicon)
- Création de la mini-charte graphique
- Préparation du dossier de livraison

## Outils

| Outil | Usage |
|-------|-------|
| Figma | Maquettes, prototypes, design system |
| Adobe Illustrator | Logos, illustrations vectorielles |
| Adobe Photoshop | Retouche photo, compositions |
| Canva | Supports rapides, réseaux sociaux |

## Règles internes

- Toujours travailler en vectoriel pour les logos
- Nommer les fichiers : `client-type-version` (ex: `catech-logo-v2`)
- Versionner chaque itération (ne jamais écraser)
- Sauvegarder sur le cloud après chaque session de travail
- Présenter les créations en contexte (mockups réalistes)

## Délais standards

| Prestation | Délai |
|-----------|-------|
| Logo seul | 7-10 jours |
| Identité visuelle complète | 12-15 jours |
| Flyer / affiche | 3-5 jours |
| Carte de visite | 2-3 jours |
| Kit réseaux sociaux | 5-7 jours |
